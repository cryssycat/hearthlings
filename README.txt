Original Species Hub Template

Files included:
- index.html
- species-hub.css
- species-hub.js

How to use:
1. Open index.html in your browser to preview the hub.
2. Edit the species name and intro text in index.html.
3. Edit SPECIES_TRAITS in species-hub.js to change trait types, rarities, and roll options.
4. Edit EXAMPLE_GALLERY in species-hub.js to change gallery examples.
5. Replace the banner background by adding a URL or image path in species-hub.css under .species-banner.

Tabs included:
- Intro
- Traits
- Generator
- Gallery

Generator features:
- Rolls one trait from every trait type.
- Displays rarity beside each trait.
- Copy button copies the generated list.
